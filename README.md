> 한국·중국·일본 3개국 모바일 게임 시장을 한눈에 파악하는 통합 분석 대시보드

🔗 **한국어**: https://jycho7254-commits.github.io/game-dashboard-v2/
🔗 **중국어**: https://jycho7254-commits.github.io/game-dashboard-cn/

## 📊 주요 기능

### 7개 탭
| 탭 | 내용 |
|---|------|
| **종합** | 3개국 매출 Top10 + 시장 점유율 + 트렌드 차트 |
| **매출 순위** | 국가별 Top50 게임 매출 추정 (USD + 원화 병기) |
| **여론 분석** | 국가별 커뮤니티 여론 지수 (DC/GP/YT/Bilibili/5ch/Twitter) |
| **인플루언서** | 유튜버 랭킹 + 실시간 방송 (SOOP/치지직 분할 표시) |
| **마케팅** | 게임별 마케팅 활동 감지 (바이럴/UA/오프라인) |
| **뉴스** | 3개국 게임 뉴스 (한국/CN gamersky/일본) |
| **캘린더** | 게임 업데이트 일정 |

## 💰 매출 추정 방식

### 기준

- **환율**: 1,500원 / $1
- **주기**: 월간 추정 매출

### 랭킹별 매출표 (월매출)
| 순위 | 🇰🇷 한국 (GP+iOS) | 🇯🇵 일본 (GP+iOS) | 🇨🇳 중국 (iOS only) |

| 10위 | 35억 ($2.3M) | 80억 ($5.3M) | 600억 ($40M) |
| 20위 | 15억 ($1.0M) | 40억 ($2.7M) | 250억 ($17M) |
| 50위 | 3억 ($200K) | 12억 ($800K) | 50억 ($3.3M) |

### 출처 태그
| 국가 | 태그 | 이유 |
|------|------|------|
| 한국 | **GP+iOS** | 구글플레이 60~70% + 앱스토어 30~40% |
| 일본 | **GP+iOS** | 앱스토어 55~65% + 구글플레이 35~45% |
| 중국 | **iOS** | 구글플레이 중국 서비스 없음 |

### 보간 방식
랭킹 사이값은 로그 보간으로 계산:
```
t = (log(rank) - log(r1)) / (log(r2) - log(r1))
매출 = v1 × (v2/v1)^t
```

## 🔧 기술 아키텍처

### 구조
```
game_dashboard_v2/
├── templates/index.html    # 메인 템플릿 (7탭, Chart.js, 인라인 데이터)
├── build_dashboard.py       # 빌드 스크립트 (JSON → placeholder 치환)
├── data/                    # 데이터 폴더
│   ├── kr_rankings.json     # 한국 Top50 랭킹
│   ├── cn_rankings.json     # 중국 Top50 랭킹
│   ├── jp_rankings.json     # 일본 Top50 랭킹
│   ├── yt_full_data.json    # YouTube 영상 데이터
│   ├── all_influencers.json # 인플루언서 231명
│   ├── softc_data.json      # 소프트콘 실시간 (fetch용)
│   ├── chzzk_live.json      # 치지직 API 실시간
│   └── ...
├── scripts/                 # 데이터 수집 스크립트
└── index.html               # 빌드 결과물 (배포용)
```

### 빌드 방식
- **인라인 방식**: 14개 JSON을 `index.html`에 직접 주입 (fetch 불필요, 단일 파일 배포)
- **압축**: `json.dumps(data, ensure_ascii=False, separators=(',',':'))`로 경량화
- **배포**: GitHub Pages (KR/CN 2개 repo)

### 실시간 데이터
- **치지직 API**: `api.chzzk.naver.com` (인증 없이 가능)
- **SOOP**: 소프트콘 CDP 크롤링 (Chrome 9223 포트)
- **소프트콘**: CDP Chrome에서 Playwright 연결 → raw 텍스트 → Python 파싱
